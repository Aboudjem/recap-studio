Install in Claude Code

One command from the 10x marketplace, then run /recap in any session.

claude plugin marketplace add Aboudjem/10x
claude plugin install recap-studio@10x
/recap "What is React Server Components"

Use it anywhere (CLI and MCP)

Link the skills into any AI CLI with the one-line installer, or run the local MCP server as the universal fallback. The CLI is not on npm yet, so clone the repo and build it first.

curl -fsSL https://raw.githubusercontent.com/Aboudjem/recap-studio/main/install.sh | bash -s codex
git clone https://github.com/Aboudjem/recap-studio && cd recap-studio
pnpm install && pnpm -w build
node packages/cli/dist/index.js render content.json

What you get on the page

Self-contained offline HTML

One file, all CSS inlined, zero JavaScript, zero external requests. Opens via file:// on a plane.

Hero answer, not an intro

One-sentence answer up top, then three takeaways in large type above the fold.

Inline SVG concept maps

Diagrams are hand-authored inline SVG, never decoration, never a hosted image.

Cited sources

Every key idea, comparison row, and timeline item references a source map entry.

Session recap from git

Point it at a git diff plus commits and it builds a recap of what changed and why.

Dark-mode, mobile-first

Sans-serif type, tasteful gradients, reduced-motion safe, readable on a phone.

Honest by design

The validation score is a deterministic heuristic check (structure, citation presence, word counts, secret and fluff scans). It does not fetch sources and does not call an LLM. The full multi-agent review (13 specialist agents) runs only when you use /recap inside Claude Code. Nothing phones home, no deploys, no emails, no secret writes, all off until you opt in.
